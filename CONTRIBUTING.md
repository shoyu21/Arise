# Contributing to ARISE

Thank you for your interest in ARISE!

---

## Current Status

**ARISE is currently in a closed development phase.**

The source code is publicly visible for transparency, but the project does not accept external contributions at this time.

---

## What You CAN Do

### Report Issues
If you encounter:
- Bugs or errors in the tool
- Inaccurate clinical information
- Broken functionality
- User interface problems

Please open an Issue in this repository with:
1. Description of the problem
2. Steps to reproduce
3. Expected vs actual behavior
4. Device and browser information

### Provide Clinical Feedback
We welcome clinical feedback regarding:
- Evidence accuracy
- Clinical workflow integration
- Missing considerations
- Special population guidance

Please provide feedback through GitHub Issues with the label "clinical-feedback".

### Request Features
You may suggest features by opening an Issue with:
- Description of the proposed feature
- Clinical use case
- How it would improve patient care

---

## What You CANNOT Do

### Code Contributions
- Pull requests are not accepted
- Code modifications should not be submitted
- Forks should not create derivative works

### Why?
ARISE is proprietary software under active development. The intellectual property rights require controlled development to:
- Maintain clinical accuracy
- Ensure consistent quality
- Protect commercial interests
- Control the evidence synthesis process
